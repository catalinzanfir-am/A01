# switch

Syntax is 

<pre>
switch (expr) {
    case '':
        break;
    case '':
        break;
    default:
}
</pre>

&nbsp;   
&nbsp;   
&nbsp;   
&nbsp;   
&nbsp;   

**Examples:**

1:
<pre>
$i = 0;
switch ($i) {
    case 0:
        echo "i equals 0";
        break;
    case 1:
        echo "i equals 1";
        break;
    case 2:
        echo "i equals 2";
        break;
}
</pre>

output:
<pre>
0 equals 0
</pre>
